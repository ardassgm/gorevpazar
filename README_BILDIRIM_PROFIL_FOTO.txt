GörevPazar Güncelleme: Bildirim Sistemi + Profil Fotoğrafı

Eklenenler:
- Kullanıcı profil fotoğrafı ekleme/güncelleme
- Panelde profil fotoğrafı gösterimi
- Navbar üzerinde profil fotoğrafı ve okunmamış bildirim sayısı
- Bildirimler panel sekmesi
- Tüm bildirimleri okundu yapma
- Yeni başvuru, başvuru kabulü, teslim, görev onayı, revizyon, uyuşmazlık, yorum, mesaj, destek cevabı, bakiye onayı ve çekim durumu için bildirim oluşturma

Test:
1) npm install
2) npm start
3) İki kullanıcıyla test edin:
   - A görev açsın
   - B başvursun
   - A panelde bildirim görmeli
   - A başvuruyu kabul etsin
   - B panelde bildirim görmeli
   - B teslim etsin
   - A bildirim görmeli
4) Panel > Profil & Güvenlik kısmından profil fotoğrafı ekleyin.

Not:
- Profil fotoğrafı için 1 MB altı görsel seçin.
- Bildirimler veritabanında notifications tablosunda tutulur.
