GörevPazar - Shopier Bakiye Güncellemesi

Eklenenler:
- Kullanıcı paneline "Bakiye Yükle" sekmesi eklendi.
- Shopier bakiye paketleri eklendi: 50, 100, 150, 250, 350, 500 TL.
- Kullanıcı Shopier'de ödeme yaptıktan sonra ödeme bildirimi gönderebilir.
- Admin panelde "Shopier Bakiye Bildirimleri" alanı eklendi.
- Admin bildirimi onaylarsa tutar kullanıcının bakiyesine eklenir.
- Admin bildirimi reddederse bakiye eklenmez.
- Ücretli görev yayınlamak için artık demo ödeme yerine kullanıcının bakiyesinden ödeme alınır.

Test Akışı:
1. Kullanıcı hesabıyla giriş yap.
2. Panel > Bakiye Yükle sekmesine gir.
3. Bir Shopier paketine tıkla ve ödeme bildirimi gönder.
4. Admin hesabıyla giriş yap: admin@gorevpazar.com / admin123
5. Admin Paneli > Shopier Bakiye Bildirimleri kısmından bildirimi onayla.
6. Kullanıcının bakiyesinin arttığını kontrol et.
7. Kullanıcı ücretli görev açsın.
8. "Bakiyemden Öde ve Yayınla" butonuna bassın.
9. Bakiye düşmeli ve görev yayına alınmalı.

Not:
Shopier otomatik callback entegrasyonu yapılmadı. Beta için manuel admin onay sistemi eklendi.
