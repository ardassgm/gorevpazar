Mikro görev güncellemesi:
- Ücretli görev minimum bütçesi 0,25 TL yapıldı.
- Admin için günlük ücretsiz görev limiti kaldırıldı.
- Görev açma ekranına fiyat önerileri ve hızlı bütçe butonları eklendi.
- Ana sayfaya mikro görev fiyat bilgileri eklendi.
- Para gösterimleri kuruşlu tutarları destekleyecek şekilde düzenlendi.
- Minimum para çekme limiti 50 TL olarak korundu.

Canlıya almak için:
1) npm install
2) npm start ile local test
3) git add .
4) git commit -m "mikro gorev fiyatlari eklendi"
5) git push
